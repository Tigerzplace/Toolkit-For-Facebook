Enable / Disable "Seen" tick for Facebook messages

Enabling this tool will hide "Seen" tick for messages sent using Facebook desktop site allowing you to read messages on Facebook's desktop site without letting the recipient know what you have read it.
After enabling this tool a script will be executed in background that will block certain request sent to Facebook servers that are used for indicating a message is read or unread.
After enabling this tool you will not be able mark a message as read or unread as long as the tool is enabled.
